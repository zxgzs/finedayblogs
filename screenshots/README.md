## 截图步骤

### 1. 启动开发服务器
```bash
cd d:/boke
npm run dev
```

### 2. 访问博客
打开浏览器访问：http://localhost:5173

### 3. 开始截图
按照 `SCREENSHOT_GUIDE.md` 中的指导逐个截取页面

### 4. 保存截图
将截图保存到此文件夹，文件名与上述清单一致

## 截图工具推荐

- **Windows**: Snipaste (https://zh.snipaste.com/)
- **Mac**: Snipaste 或 Xnip
- **浏览器自带**: Win+Shift+S (Windows) 或 Cmd+Shift+4 (Mac)

## 注意事项

1. 所有截图使用 PNG 格式
2. 单张截图大小建议控制在 500KB 以内
3. 保持截图清晰度和美观度
4. 确保截图内容不包含敏感信息
5. 建议使用 100% 浏览器缩放比例
