## 截图工具推荐

### Windows
1. **Windows 系统截图**：`Win + Shift + S`
2. **Snipaste**：https://zh.snipaste.com/ （推荐）
3. **ShareX**：https://getsharex.com/

### Mac
1. **Mac 系统截图**：`Cmd + Shift + 4`
2. **Snipaste**：https://zh.snipaste.com/
3. **Xnip**：https://xnip.com/

### 通用
- **Chrome DevTools**：F12 → Ctrl+Shift+P → 输入 "Capture screenshot"
- **Firefox**：开发者工具 → 网络监控 → 右键 → 截图

## 截图技巧

1. **浏览器缩放**：确保浏览器缩放为 100%
2. **清理缓存**：截图前清除浏览器缓存，确保最新效果
3. **关闭干扰**：关闭浏览器扩展，避免干扰截图
4. **选择合适时间**：选择页面加载完成后的稳定状态
5. **保持一致性**：所有截图使用相同的浏览器和缩放比例
6. **图片格式**：建议使用 PNG 格式，质量更好
7. **文件大小**：单张截图建议控制在 500KB 以内
8. **图片尺寸**：宽度建议 1200px-1600px，高度根据内容调整

## 截图步骤

### 准备工作
1. 启动开发服务器：`npm run dev`
2. 在浏览器中打开：http://localhost:5173
3. 准备截图工具

### 截图流程
1. 访问需要截图的页面
2. 等待页面完全加载
3. 调整浏览器窗口大小（如果需要）
4. 触发交互效果（如悬停、点击等）
5. 使用截图工具截取
6. 保存到 `d:/boke/screenshots/` 目录
7. 按照上述文件名保存

## 注意事项

1. **不要泄露隐私**：确保截图不包含个人信息、API密钥等
2. **清理测试数据**：如有测试数据，建议使用真实或模拟数据
3. **保持最新**：截图应反映当前版本的最新功能
4. **质量第一**：截图清晰、美观，能吸引读者
5. **统一风格**：所有截图保持一致的视觉风格
6. **添加标注**：可以在截图上添加必要的文字标注（可选）
