# 📚 如何创建文章系列

## 🎯 快速开始

### 方法一：通过管理后台创建（推荐）

1. **访问管理后台**
   - 在浏览器中打开博客
   - 点击导航栏的"管理后台"或访问 `/admin`
   - 输入管理员信息登录

2. **进入系列管理**
   - 在管理后台的标签页中，点击"系列管理"
   - 你会看到所有现有的系列列表

3. **创建新系列**
   - 点击左上角的"创建系列"按钮
   - 填写以下信息：

   **必填项：**
   - **系列标题**：例如"Vue 3 完全指南"
   - **系列描述**：例如"从入门到精通的 Vue 3 系列教程"

   **可选项：**
   - **封面图片**：输入图片URL（建议尺寸：1200x630px）
   - **状态**：
     - `草稿` - 仅自己可见
     - `连载中` - 公开显示，持续更新
     - `已完结` - 公开显示，不再更新

4. **保存系列**
   - 点击"创建"按钮
   - 系列创建成功！

5. **管理系列文章**
   - 在系列卡片上点击"管理文章"
   - 从左侧"可用文章"中勾选要添加的文章
   - 在右侧"已选文章"中拖拽调整顺序
   - 点击"保存排序"

---

### 方法二：直接访问系列页面

1. **访问系列列表**
   - 访问 `/series` 页面
   - 查看所有已发布的系列

2. **创建系列**
   - 点击页面上的"创建系列"按钮（如果有）
   - 或者返回管理后台创建

## 📋 系列管理操作

### 编辑系列
1. 在管理后台的"系列管理"标签页
2. 找到要编辑的系列
3. 点击卡片上的"编辑"按钮
4. 修改信息后点击"更新"

### 删除系列
1. 在系列卡片上点击"删除"按钮
2. 确认删除操作
3. ⚠️ **注意**：删除系列不会删除其中的文章

### 查看系列
- 点击卡片上的"查看"按钮
- 或直接访问 `/series/:id` 页面

## 🎨 系列封面图片建议

### 推荐尺寸
- **宽度**：1200px
- **高度**：630px
- **格式**：JPG、PNG、WebP
- **大小**：建议小于 500KB

### 图片来源
1. **免费图库**
   - Unsplash (https://unsplash.com)
   - Pexels (https://pexels.com)
   - Pixabay (https://pixabay.com)

2. **AI 生成**
   - DALL-E 3
   - Midjourney
   - Stable Diffusion

3. **自定义设计**
   - Canva (https://canva.com)
   - Figma
   - Photoshop

### 使用方法
1. 上传图片到图床服务（如阿里云OSS、七牛云）
2. 获取图片URL
3. 在创建/编辑系列时粘贴URL

## 📝 创建系列最佳实践

### 1. 规划系列结构
```
系列名称：[技术栈] [主题] 指南
例如：
- Vue 3 响应式编程指南
- TypeScript 高级类型系统
- CSS 现代布局技术
```

### 2. 系列描述撰写
- ✅ 清晰说明系列目标
- ✅ 列出主要知识点
- ✅ 说明适合人群
- ✅ 预计学习时长

**示例：**
```
本系列将带你深入理解 Vue 3 的响应式系统，
从基础概念到源码分析，适合有一定 Vue 基础
的开发者，预计学习时长：8小时
```

### 3. 文章排序原则
- 按照学习逻辑递进
- 从易到难，循序渐进
- 每篇文章建立在前面内容的基础上

### 4. 系列状态管理
- **草稿**：还在规划阶段
- **连载中**：正在持续更新，吸引关注
- **已完结**：全部完成，可以推荐

## 💡 常见问题

### Q: 如何删除系列中的文章？
A: 在管理后台点击"管理文章"，移除不需要的文章即可。

### Q: 文章可以属于多个系列吗？
A: 可以，同一篇文章可以添加到多个系列中。

### Q: 如何更改系列文章的顺序？
A: 在"管理文章"中拖拽右侧列表中的文章，然后保存排序。

### Q: 删除系列会影响文章吗？
A: 不会，删除系列只是删除系列本身，其中的文章会保留。

### Q: 系列封面不显示怎么办？
A: 检查图片URL是否正确，确保图片可以被公开访问。
